HOW IT WORKS

Semantic matching,
not string matching.

Exact-match caching fails when users rephrase the same question. CacheLayer understands meaning.

01
Query arrives
Your application sends a request to CacheLayer instead of directly to OpenAI or Anthropic. Nothing else changes.
02
Semantic embedding
The query is converted to a 384-dimensional vector using a sentence transformer model. Similar meanings produce nearby vectors.
03
Similarity search
Cosine similarity is computed against all cached entries. If a match scores above the threshold, the cached response is returned instantly.
04 — CACHE HIT
Zero tokens spent. Response in ~8ms.
The cached response is returned in the standard OpenAI format. Your code sees no difference. You spend nothing.
INTEGRATION
